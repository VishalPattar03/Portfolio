AI - Smart Diagnosis

Smart Diagnosis tool based on Convolution Neural Network to classify critical Diseases related to Kidney, Brain-Tumor, Lungs and Tuberculosis.

Live Link

Project Overview

AI Smart Diagnosis is a Streamlit web application designed to provide smart diagnosis for various diseases using CT scan images. It features secure login authentication and allows users to upload CT scan images for the diagnosis of brain tumors, tuberculosis, kidney disease, and lung disease. The application employs machine learning models to analyze the images and deliver accurate diagnoses. Additionally, it suggests relevant medications and precautions based on the diagnosed condition.

The app emphasizes consulting a physician for medical advice, reinforcing that the provided information is supplementary. The project is available for live demo and open-source contributions.
